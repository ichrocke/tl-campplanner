Feature Ideas for Camp Planner
==============================

STATUS: [x] = implemented, [ ] = open, [~] = partially done


## Layers & Organization
[x] Multiple layers per camp site
[x] Show/hide layers, lock layers, reorder
[x] Move objects between layers
[x] Active layer indicator
[ ] Layer opacity (transparency per layer)
[ ] Layer colors (visual tint for identification)
[ ] Merge layers
[ ] Flatten all layers into one


## Infrastructure & Utilities
[x] Pipe/cable tool with configurable thickness and color presets
[x] Safety symbols (First Aid, Fire Ext, Gas, Electric, etc.)
[ ] Electrical connection points with capacity labels (16A, 32A, CEE)
[ ] Water tap symbols with flow rate/direction
[ ] Drainage/sewage line markers (dashed pipe style)
[ ] Light pole positions with illumination radius circles
[ ] WiFi access point range circles
[ ] Power distribution diagram (which box feeds which tent)
[ ] Cable length calculator along pipe routes


## Measurements & Planning
[x] Persistent guide lines with distance labels
[x] Edge lengths on ground areas
[x] Area in m² on ground areas
[ ] Automatic distance labels between nearest objects
[ ] Area utilization percentage (tents vs. total ground area)
[ ] Person capacity calculator (sum tent capacities)
[ ] Minimum distance violation highlighting (permanent, not just while dragging)
[ ] Path routing / walkway planning between tents
[ ] Vehicle access width checker (minimum 3.5m)
[ ] Slope/grade indicators for drainage planning
[ ] Sun position overlay (shadow casting at specific time/date)


## Collaboration & Sharing

[ ] QR code generation for sharing plan URL

[ ] Team roles (view-only, edit, admin)
